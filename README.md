# ML Notes

I am reading Hands-on-ML during summer 2026 preparing for my internship interviews. I created a series of useful notes while going over the texts which include example code implementations and their explanations for a variety of ML models.

This notes can be viewed as a summary of the codes covered in the book, with some modifications by myself for better understanding.

The goal of this repository is to document my learning process and build a clear reference for understanding both the theory and practical implementation of common Machine Learning algorithms.

## About This Repository

Topics may include:

- writing a pipeline for data preprocessing
- k-folds cross validation
- gridsearch
- precision, recall, f1 scores, confusion matrix, ROC
- linear regression (SVD + gradient descent approaches)
- polynomial regression
- learning curves to detect underfitting/overfitting
- ridge regression and lasso regression to address overfitting
